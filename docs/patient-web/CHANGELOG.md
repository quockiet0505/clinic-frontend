### UI & UX Improvements
- **Profile Layout Refactor**: 
  - Restored the profile banner and header layout.
  - Reorganized profile information, grouping inputs effectively (e.g., date of birth and phone number on the same row) and removing explicit sections to streamline the form.

  - Formatted labels into standard case rather than uppercase.
